## Fun With CSS

This repository focuses on creating nice CSS effects.

* [0-styles.css](https://github.com/gwendalminguy/holbertonschool-Fun-with-CSS/tree/main/0-styles.css), a CSS file to create an image sprite.

* [1-styles.css](https://github.com/gwendalminguy/holbertonschool-Fun-with-CSS/tree/main/1-styles.css), a CSS file to create a text underline animation.

* [2-styles.css](https://github.com/gwendalminguy/holbertonschool-Fun-with-CSS/tree/main/2-styles.css), a CSS file to create a toggle switch effect.

* [3-styles.css](https://github.com/gwendalminguy/holbertonschool-Fun-with-CSS/tree/main/3-styles.css), a CSS file to create a burger menu effect.
